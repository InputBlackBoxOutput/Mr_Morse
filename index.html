What is morse code?

Morse Code was used to send messages over long distances. Morse code messages can be sent using light or by pulses. A telegraph sends pulses in the form of electrical current based on the message that was "tapped out" using the telegraph key. Telegraph operators would key messages using a series of clicks based on the spelling of the words of the messages. An operator on the receiving end would hear the clicks and translate them back into words. With Morse Code, each letter of the alphabet was translated into short and long signals (also known as dots and dashes). The pulse length of a dash is equal to the pulse length of three dots.
