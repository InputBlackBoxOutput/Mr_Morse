What is Morse code?

Morse code is an obsolete encoding scheme. It is named after Samuel Morse, inventor of the telegraph. It was developed by him for use during telegraph transmission. In Morse code, every supported character is substituted by a series of ‘dots’ and ‘dashes’ or sometimes just singular ‘dot’ or ‘dash’ and vice versa. An example is shown below.

Morse code token ‘dot’ is equivalent to binary 0 and Morse code token ‘dash’ is equivalent to the binary 1. Morse code is still used today by armies all around the world when all other communications fail.

Why Morse code has different lengths for different characters?

More frequently used characters (such as vowels) have less number of dots or dashes while characters used less frequently have more. This is done in order to improve the telegraph transmission speed.

Why Morse code is not used today?

Morse code is not used because of the following reasons:

  • It does not support all characters from all the languages in the world.
  • It is not prefix code which means that a pause is required after every letter is sent while a longer pause is required after every word. This leads to large transmission time.

Fun facts about Morse code

  • The old Nokia text message tone (Remember it?) is morse code for SMS.
  • One of the most common signals is the distress signal SOS. Three dots, three dashes and three dots. This signal is internationally recognised by treaty.
  • Morse code can be transmitted by light, radio waves or sound. Wowza that's a lot of communication mediums!
